# ZAD - منصة التبرع بالطعام 🍽️

> A Flutter & Firebase mobile application connecting food donors, associations, and volunteers to reduce food waste in Tlemcen, Algeria.

## 📱 About the App

ZAD is a Licence graduation project developed at the University of Tlemcen, presented within the i2E entrepreneurship program. The app aims to reduce food waste by creating a bridge between food donors and those in need, managed through verified associations and volunteers.

## ✨ Features

- 🔐 Multi-role authentication (Donor, Association, Volunteer, Admin)
- 📍 Real-time GPS tracking for food pickups
- 💬 In-app chat between users
- 🎁 Points, coupons & QR rewards system
- 🖼️ Image uploads via Cloudinary
- 💳 Financial donations (CCP / BaridiMob)
- 🌐 Web-based Admin Panel
- ✅ Pickup code verification system

## 🛠️ Tech Stack

| Technology | Usage |
|------------|-------|
| Flutter / Dart | Mobile & Web frontend |
| Firebase Firestore | Real-time database |
| Firebase Auth | Authentication |
| Firebase Storage | File storage |
| Cloudinary | Image management |
| Geolocator | GPS tracking |
